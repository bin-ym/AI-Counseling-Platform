# Setup Instructions
## Tools
- Git: 2.47.0
- Python: 3.13.3
- Node.js: 20.18.0
- PostgreSQL: 16.x
- VS Code
- Figma (web)

## Steps
1. Install tools (see above versions).
2. Set up Python virtual environment: `python -m venv venv` and `.\venv\Scripts\activate`
3. Create database: `psql -U postgres -c "CREATE DATABASE counseling_db;"`
